## PharmaCity

PharmaCity, şehir bazlı nöbetçi eczaneleri listeleyen ve kullanıcıları temel sağlık konularında güvenli şekilde yönlendiren bir web uygulamasıdır.  
Projede ai sohbet asistanı bulunur.

---

## Özellikler

-  Şehre göre eczane listeleme  
-  Nöbetçi eczane desteği  
-  Admin panel (şehir / eczane yönetimi)  
-  Keysiz chatbot  
-  Kurulumsuz SQLite modu

---

## Teknolojiler

- Node.js + Express  
- Vanilla JS + HTML  
- SQLite (default) / MySQL (opsiyonel)

---

## Çalıştırma

```bash
cd "pharmacity backend"
npm install
npm start
